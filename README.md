# 🌐 Zypo Mesh Network

**Zypo** — это пуленепробиваемый, децентрализованный P2P-протокол следующего поколения, объединяющий в себе зашифрованную сеть передачи данных, распределенную DNS-систему и автономную P2P-экономику.

## 🚀 Основные возможности

### 1. Децентрализованная P2P Экономика (ZPCN)
В отличие от традиционных систем, Zypo не имеет "центрального банка".
*   **Локальные кошельки:** Каждое устройство само хранит свой баланс и историю транзакций.
*   **Ed25519 Подписи:** Каждая транзакция подписывается закрытым ключом устройства и проверяется получателем локально.
*   **Автономность:** Платежи работают даже если Командный Центр временно недоступен.
*   **Микроплатежи:** Система State Channels позволяет оплачивать VPN-трафик побайтово без нагрузки на сеть.

### 2. Bullet-Proof VPN & Proxy
Полноценный P2P-туннель поверх libp2p с двусторонним шифрованием. 
*   **SOCKS5 & Forward Proxy (HTTPS):** Встроенные локальные прокси-серверы (SOCKS5 на порту `8910`, HTTP Proxy на `8903`) для надежного роутинга трафика. Поддерживается полноценный HTTPS (CONNECT) через локальный TLS MITM, что позволяет открывать зашифрованные сайты в обычных браузерах (Chrome/Firefox).
*   **Опциональный TUN (Linux/macOS):** При наличии прав администратора можно поднять виртуальный сетевой интерфейс для прозрачного перехвата и маршрутизации всего TCP-трафика ОС.
*   **Windows Fallback:** Если драйверы TUN отсутствуют (поведение по умолчанию для Windows), нода прозрачно и безопасно откатывается в SOCKS5-only режим.
*   **DHT Discovery:** Поиск VPN-провайдеров происходит через распределенную хеш-таблицу (Kademlia), исключая центральные реестры.

### 3. Распределенная DNS & Безопасный Хостинг
*   **Zypo DNS:** Система именования в зонах `.zypo`, `.mesh`, `.rus` и др.
*   **Oracle Verification & Mesh-Fallback:** Все официальные записи подписываются Командным Центром (Oracle). В случае падения Центра, сеть переходит в децентрализованный "Mesh-Only" режим и продолжает работу на базе P2P DHT.
*   **Скрытые сервисы (Secure Sandbox):** Хостинг сайтов внутри сети без публичного IP. Встроена жесткая защита от Path Traversal — выйти за пределы директории сайта невозможно.

### 4. Economy (Mesh-Finance)
Зачаток внутренней экономики (ZPCN). Платите за VPN-трафик токенами, получайте токены за раздачу интернета.

## 🛡️ Модель доверия и Безопасность

### Как подтверждается подлинность глобального реестра (Oracle)?
1.  **Trust-on-First-Connect (TOFC):** При первом успешном подключении к официальной бутстрап-ноде (указанной в конфиге), клиент извлекает Ed25519 публичный ключ.
2.  **Oracle Persistence:** Этот ключ сохраняется в `data/oracle.pub`.
3.  **Signature Validation:** Все DNS-записи в DHT, претендующие на официальный статус, должны быть подписаны этим ключом.
4.  **Decentralization:** Если узел-оракул скомпрометирован или уходит в оффлайн, сеть продолжает функционировать, а балансы проверяются полностью децентрализованно.

## 📂 Структура проекта

*   `dot-zypo-common`: Ядро. Либы P2P, DHT, маршрутизация, DNS-логика, хостинг.
*   `dot-zypo-control`: Command Center (Нода-Координатор). Выступает как DNS Oracle и публичный Relay-хаб.
*   `dot-zypo-client`: Клиентский демон. Поднимает TUN-интерфейс, SOCKS5 прокси, HTTPS Forward Proxy для браузера.
*   `dot-zypo-browser`: Графический интерфейс пользователя и браузер на базе Electron.
*   `dot-zypo-yan`: Поисковый движок и краулер для скрытой Mesh-сети.
*   `dot-zypo-server`: (Устарел) Бывший сигнальный сервер WebRTC. Подлежит удалению.

## 🛠️ Быстрый запуск

### 1. Клиентская нода (Client)
Для подключения к сети вам потребуется адрес любой доверенной Bootstrap-ноды. По умолчанию клиент попытается использовать встроенные адреса:
```bash
go run dot-zypo-client/main.go
```
Если вы хотите явно указать основную координационную ноду:
```bash
go run dot-zypo-client/main.go -bootstrap /ip4/213.171.27.234/tcp/8900/p2p/12D3KooWPyuxuDGwsnBB6sKvCarjfRimWhNvvXaAhYnGkesBFbEf
```

### 2. Запуск Браузера
После того, как клиентская нода запустится и перейдет в режим `Synced`, вы можете открыть Zypo Browser:
```bash
cd dot-zypo-browser
npm install
npm start
```

### 3. VPN Провайдер (Public Relay/Exit Node)
Если вы хотите зарабатывать `ZPCN`, выступая в роли VPN-провайдера, запустите серверный узел на VPS с публичным IP:
```bash
go run dot-zypo-server/main.go
```
